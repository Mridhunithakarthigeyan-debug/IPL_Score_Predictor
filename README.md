This project predicts the final first innings score of an IPL match using Machine Learning and historical IPL data. The model is built using the XGBoost Regressor 
algorithm and predicts scores based on match statistics after the first 6 overs, such as runs, wickets, and run rate. The project includes data preprocessing,
feature engineering, model training, evaluation, and a simple frontend interface for live score prediction. Technologies used include Python, Pandas, Scikit-learn,
XGBoost, Flask, HTML, CSS, and JavaScript.
